NerdMiner Web Flasher — AxeHub Edition

source on GitHub ↗
Requirements: Chrome / Edge / Opera (Web Serial API). Firefox and Safari are NOT supported. Connect ESP32 via USB and click Connect & Flash. For ESP32-S3 use the USB port (not UART).
Default chain: BC2 (BitcoinII). To switch to BTC after flashing, use the API: curl -X POST http://<ip>/api/axehub/v1/coin -d 'ticker=BTC' -H 'X-Axehub-Compat: 1'.
